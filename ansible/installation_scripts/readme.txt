# Installation of CQube

For Linux: 
---------

- Open Terminal
- Open the directory of downloaded folder 
  	-> cd <path>
- Run the install.sh shell script file using sudo command
	-> sudo ./install.sh

Observations:

- while installation
	- see ansible installing
	- see Installing jdk
	- see Installing python3 and pip
	- see Downloading nifi
	- see Extracting nifi to /opt
	- see Creating S3 Input bucket
	- see Creating S3 Output bucket
	- see Installing and Configuring sftp
	- see the message 'CQube installed successfully!!' 

- if anywhere fails
	- see the failed message in red color


For Windows:
-----------

To install in Windows Machine use
./install.bat
